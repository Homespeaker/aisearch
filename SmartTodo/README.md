# SmartTodo

SmartTodo - это простой консольный TODO-лист на Python с небольшой "умностью".
Программа считает приоритет задач и подсказывает, что лучше сделать прямо сейчас.

## Что умеет

- добавлять задачи;
- показывать все активные задачи в порядке приоритета;
- советовать лучшую задачу на текущий момент;
- отмечать задачи выполненными;
- сохранять данные в `tasks.json`.

## Формула приоритета

```text
приоритет = срочность * 2 + сложность + бонус за короткое время
```

Бонус за короткое время:

- до 15 минут: `+3`
- до 30 минут: `+2`
- до 60 минут: `+1`
- больше 60 минут: `+0`

## Запуск

```bash
python main.py
```
